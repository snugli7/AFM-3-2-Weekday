# 돼지고기 계란볶음밥

## 소개

고소한 계란과 잘 볶아진 돼지고기가 어우러진 간단하고 맛있는 한 그릇 요리입니다.

---

## 재료 (1인분)

| 재료 | 분량 |
|------|------|
| 밥 (찬밥 권장) | 1공기 (200g) |
| 돼지고기 (앞다리살 또는 목살) | 100g |
| 계란 | 2개 |
| 대파 | 1/2대 |
| 양파 | 1/4개 |
| 당근 | 1/4개 |
| 식용유 | 2큰술 |
| 참기름 | 1/2큰술 |
| 소금 | 약간 |
| 후추 | 약간 |

### 돼지고기 밑간

| 재료 | 분량 |
|------|------|
| 간장 | 1큰술 |
| 맛술 (또는 청주) | 1큰술 |
| 다진 마늘 | 1/2큰술 |
| 후추 | 약간 |

---

## 만드는 법

### 1단계: 재료 준비

1. **돼지고기**를 작은 큐브 모양(1cm)으로 썰어 밑간 재료에 10분간 재워둔다.
2. **양파**, **당근**은 작은 큐브 모양으로 썬다.
3. **대파**는 송송 썬다.
4. **계란** 2개를 볼에 깨서 소금 한 꼬집을 넣고 잘 풀어둔다.

### 2단계: 돼지고기 볶기

1. 팬에 식용유 1큰술을 두르고 **중강불**로 달군다.
2. 밑간한 돼지고기를 넣고 **겉면이 노릇해질 때까지** 약 3~4분 볶는다.
3. 양파, 당근을 넣고 **1~2분** 더 볶은 후 접시에 덜어둔다.

### 3단계: 계란 볶기

1. 같은 팬에 식용유 1큰술을 추가로 두른다.
2. 풀어둔 계란을 넣고 **젓가락으로 빠르게 저으면서** 반숙 스크램블을 만든다.
3. 계란이 70% 정도 익으면 바로 다음 단계로 넘어간다.

### 4단계: 밥 볶기

1. 반숙 계란 위에 **찬밥**을 넣고 주걱으로 꾹꾹 눌러가며 풀어준다.
2. **강불**로 올리고 밥알이 하나하나 분리되도록 볶는다 (약 2~3분).
3. 덜어둔 돼지고기와 야채를 다시 넣고 잘 섞는다.

### 5단계: 간 맞추기 & 마무리

1. **간장 1/2큰술**, 소금, 후추로 간을 맞춘다.
2. 대파를 넣고 30초간 볶는다.
3. 불을 끄고 **참기름 1/2큰술**을 둘러 마무리한다.

---

## 꿀팁

- **찬밥**을 사용해야 밥알이 잘 분리되어 볶음밥이 파슬파슬해진다.
- 계란을 **완전히 익히지 않고** 반숙 상태에서 밥을 넣어야 밥알을 계란이 코팅하면서 황금빛 볶음밥이 된다.
- **강불**에서 빠르게 볶아야 식감이 좋다.
- 기호에 따라 **김가루**, **깨** 등을 올려 먹으면 더 맛있다.
