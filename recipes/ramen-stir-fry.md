# 라면 볶이 (볶음라면)

**난이도:** 쉬움 | **조리 시간:** 15분 | **인원:** 1인분 | **재료 매칭률:** 68%

> 삶은 라면에 돼지고기와 채소를 볶아낸 볶음라면. 유통기한 임박한 돼지고기(~6/24)를 활용하기 좋습니다.

## 사용 재료

| 재료 | 수량 | 비고 |
|---|---|---|
| 라면 | 1봉 | 스프 1/2만 사용 |
| 돼지고기 | 100g | 얇게 썰기 |
| 양파 | 1/2개 | 채 썰기 |
| 당근 | 1/4개 | 채 썰기 |
| 계란 | 1개 | |
| 파 | 약간 | 송송 썰기 |

## 조리 순서

1. 라면을 끓는 물에 2분만 삶아 건진다 (평소보다 짧게)
2. 팬에 기름을 두르고 돼지고기를 볶는다
3. 고기가 익으면 양파, 당근을 넣고 볶는다
4. 삶은 면과 라면 스프 1/2을 넣고 섞는다
5. 간장 1큰술, 참기름 약간을 넣고 볶는다
6. 계란을 풀어 넣고 빠르게 섞는다
7. 파를 올려 마무리한다

## 팁

- 면을 너무 오래 삶으면 볶을 때 퍼지니 2분만 삶으세요
- 고추장 1큰술을 넣으면 매콤한 맛이 납니다
- 김치를 추가하면 김치볶음면이 됩니다
