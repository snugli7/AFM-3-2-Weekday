# 김치라면

**난이도:** 쉬움 | **조리 시간:** 10분 | **인원:** 1인분 | **재료 매칭률:** 72%

> 김치와 계란, 파를 넣은 든든한 김치라면. 간단하지만 깊은 맛을 내는 한 끼 식사입니다.

## 사용 재료

| 재료 | 수량 | 비고 |
|---|---|---|
| 라면 | 1봉 | 스프 포함 |
| 김치 | 1/2컵 | 잘게 썰기 |
| 계란 | 1개 | |
| 파 | 약간 | 송송 썰기 |
| 양파 | 1/4개 | 채 썰기 (선택) |

## 조리 순서

1. 냄비에 물 550ml를 넣고 끓인다
2. 물이 끓으면 라면 스프와 김치를 넣는다
3. 양파를 넣고 1분간 끓인다
4. 면을 넣고 3분간 끓인다
5. 계란을 풀어 넣고 30초 더 끓인다
6. 파를 올려 마무리한다

## 팁

- 김치를 참기름에 먼저 볶으면 감칠맛이 올라갑니다
- 두부를 한 모 썰어 넣으면 더 든든합니다
- 치즈를 올리면 매운맛이 중화됩니다
