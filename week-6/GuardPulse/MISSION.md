# MISSION.md

## 1. 앱 이름

**GuardPulse (가드펄스)**

> 소중한 사람의 건강을 지키는 실시간 생체 모니터링 서비스

---

## 2. 한 줄 미션

웨어러블 워치의 생체 데이터를 실시간으로 분석하여, 위험 상황 발생 시 가족과 전문 모니터링 담당자에게 즉시 알림을 전달함으로써 중장년층과 유병자의 건강 안전망을 구축한다.

---

## 3. 타겟 사용자

| 구분 | 설명 |
|------|------|
| **1차 사용자 (착용자)** | 중장년층, 시니어, 만성질환 유병자 (고혈압, 심장질환, 당뇨 등) |
| **2차 사용자 (보호자)** | 착용자의 가족, 보호자 |
| **3차 사용자 (B2B)** | 건강 모니터링 서비스 회사의 담당자 및 AI agent |

---

## 4. 핵심 기능 (MVP)

### 4-1. 웨어러블 데이터 수집
- 스마트폰 건강 앱(Apple Health, Samsung Health 등)에서 생체 데이터 연동
- 수집 항목: 혈압, 맥박(심박수), 산소포화도(SpO2), 부정맥 감지
- 데이터 이력 저장 및 시각화 (일별/주별/월별 추이)

### 4-2. 위험 감지 시스템
- 사용자별 맞춤 위험 범위 설정 (예: 혈압 상한/하한, 심박수 이상 기준)
- 실시간 생체 데이터 모니터링 및 이상 감지
- 위험 등급 분류 (주의 / 경고 / 위험)

### 4-3. 가족 알림
- 위험 신호 발생 시 등록된 가족에게 즉시 알림 (푸시, SMS, 앱 내 알림)
- 착용자의 현재 상태 및 위치 정보 공유
- 가족 전용 대시보드 (착용자 건강 상태 한눈에 확인)

### 4-4. 모니터링 회사 담당자 알림 (B2B 연동)
- 모니터링 서비스 회사 시스템 연동 API
- AI agent 자동 대응 프로토콜 (1차 자동 판단 및 에스컬레이션)
- 전문 담당자 알림 및 대응 이력 관리

---

## 5. 차별화 포인트

1. **다층 안전망 구조**: 가족 알림에 그치지 않고, 전문 모니터링 회사의 AI agent와 담당자까지 연결하는 3중 안전망 (착용자 본인 - 가족 - 전문 담당자)
2. **AI 기반 위험 감지**: 단순 임계값 초과가 아닌, 패턴 분석을 통한 사전 위험 예측
3. **B2B 연동 플랫폼**: 모니터링 서비스 회사가 자사 시스템에 통합할 수 있는 개방형 API 제공
4. **시니어 친화 UX**: 중장년층이 쉽게 사용할 수 있는 큰 글씨, 단순한 인터페이스, 음성 안내

---

## 6. 기술 플랫폼

| 단계 | 플랫폼 | 설명 |
|------|--------|------|
| **Phase 1 (MVP)** | 웹앱 (반응형) | Next.js 기반 PWA, 모바일 브라우저 최적화 |
| **Phase 2** | 모바일 앱 | React Native 또는 Flutter로 iOS/Android 네이티브 앱 확장 |

### 기술 스택 (예상)
- **프론트엔드**: Next.js, TypeScript, Tailwind CSS
- **백엔드**: Node.js (Express 또는 Fastify)
- **데이터베이스**: PostgreSQL (Supabase)
- **실시간 통신**: WebSocket / Server-Sent Events
- **알림**: FCM (Firebase Cloud Messaging), Twilio (SMS)
- **건강 데이터 연동**: Apple HealthKit API, Samsung Health SDK
- **AI/ML**: 이상 탐지 모델 (향후 적용)

## 9. 향후 확장 계획 (MVP 이후)

### Phase 2: 모바일 네이티브 앱
- iOS / Android 네이티브 앱 출시
- 백그라운드 실시간 데이터 동기화
- 위치 기반 긴급 서비스 연동 (119 자동 연결)

### Phase 3: AI 고도화
- 생체 데이터 패턴 학습을 통한 사전 위험 예측 (예: 심장마비 전조 증상 감지)
- 개인 맞춤 건강 리포트 자동 생성
- AI agent의 자동 상담 및 응급 대응 시나리오

### Phase 4: 생태계 확장
- 병원/의료기관 연동 (전자건강기록 EMR 연계)
- 보험사 연동 (건강 데이터 기반 보험료 할인 프로그램)
- 다양한 웨어러블 기기 호환 확대 (Fitbit, Garmin, Xiaomi 등)
- 복약 관리 및 정기 검진 알림 기능

### Phase 5: 글로벌 확장
- 다국어 지원
- 해외 건강 데이터 표준 (FHIR) 연동
- 현지 모니터링 서비스 회사 파트너십
